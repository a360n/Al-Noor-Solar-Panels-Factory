# مصنع النور للألواح الشمسية | Al Noor Solar Panels Factory

موقع تعريفي واحترافي ثنائي اللغة (عربي / إنجليزي) لمصنع **النور للألواح الشمسية (Al Noor Lab)**، تم تصميمه وتطويره بالكامل باستخدام لغات الويب الأساسية وبأعلى معايير الأداء والتجاوب مع جميع الشاشات.

A professional, bilingual (Arabic/English) responsive landing page for **Al Noor Solar Panels Factory (Al Noor Lab)**, built using vanilla HTML, CSS, and JavaScript.

---

## 🌟 المميزات الرئيسية / Key Features

* **ثنائي اللغة بالكامل (Fully Bilingual)**: دعم كامل للغتين العربية (RTL) والإنجليزية (LTR) مع تبديل سلس وبدون إعادة تحميل الصفحة.
* **حاسبة إنتاج الطاقة (Solar Output Estimator)**: حاسبة تفاعلية تتيح للمستخدمين تقدير كمية الطاقة السنوية المنتجة والمساحة المطلوبة والتكلفة بناءً على نوع وعدد الألواح.
* **محاكي فحص الخلايا (EL Scan Simulator)**: جزء تفاعلي يعرض تقنية فحص التلألؤ الكهربائي (Electroluminescence) لكشف الشروخ والعيوب المجهرية في خلايا السيليكون.
* **متوافق مع الهواتف الذكية (Ultra Responsive)**: تصميم متجاوب تماماً مع الشاشات بدءاً من الهواتف الصغيرة (320px) إلى شاشات الحاسوب الضخمة.
* **سرعة فائقة (Performance Optimized)**: مبني بدون إطارات عمل ثقيلة (Vanilla Stack) لضمان سرعة تحميل استثنائية.
* **أصول بصرية متميزة (Premium Assets)**: شعار مفرغ الخلفية بدقة عالية، وصور احترافية حقيقية لخط الإنتاج والمنتجات والمشاريع.

---

## 🛠️ لغات التطوير / Tech Stack

* **HTML5**: لتقسيم وترتيب محتوى الموقع بشكل دلالي (Semantic HTML).
* **CSS3**: للتنسيقات الجمالية واستخدام متغيرات الألوان الرسمية للمصنع (التركوازي، الأسود، الذهبي) مع حواف هندسية حادة تعكس شكل الخلايا الشمسية.
* **Vanilla JavaScript**: لبرمجة الترجمات، الحاسبة التفاعلية، محاكي الفحص، وتأثيرات الظهور عند التمرير (Intersection Observer).
* **FontAwesome Icons**: للأيقونات الخدمية المتميزة.

## 🚀 التشغيل المحلي / Local Run

يمكنك تشغيل الموقع محلياً بعدة طرق:
1. **فتح الملف مباشرة**: قم بفتح ملف `index.html` في أي متصفح ويب.
2. **عبر خادم محلي (Server)**:
   ```bash
   python3 -m http.server 8000 --bind 0.0.0.0
   ```
   ثم افتح المتصفح على الرابط: `http://localhost:8000` أو من خلال عنوان IP الخاص بجهازك لتصفحه من الجوال.

## 🌐 الرفع والتشغيل عبر جيت هاب (GitHub Pages)

هذا الموقع جاهز تماماً للنشر المجاني عبر **GitHub Pages** لأن جميع مسارات الملفات فيه نسبية (Relative Paths).

### خطوات النشر / Deployment Steps:
1. قم بإنشاء مستودع جديد (Repository) على حسابك في GitHub واجعله **عاماً (Public)**.
2. ارفع جميع ملفات المشروع إلى المستودع:
   * `index.html`
   * `style.css`
   * `script.js`
   * `logo.png`
   * مجلد `assets` (يحتوي على كافة الصور).
   * ملف `README.md`
3. بعد الرفع، اذهب إلى إعدادات المستودع **Settings**.
4. من القائمة الجانبية، اختر **Pages**.
5. تحت قسم **Build and deployment**:
   * اختر **Deploy from a branch**.
   * اختر الفرع الرئيسي للمشروع (غالباً `main` أو `master`).
   * اختر المجلد الرئيسي `/ (root)`.
   * اضغط على **Save**.
6. انتظر لمدة دقيقة واحدة، وسيقوم GitHub بتوفير رابط مباشر لموقعك بصيغة:
   `https://<YOUR-USERNAME>.github.io/<YOUR-REPOSITORY-NAME>/`
